# mdmsite
mdmsite

Steps to preview:

1) Install Hugo
2) Clone Repo to drive
3) Navigate into mdmsite directory
4) run `hugo serve`
5) open http://localhost:1313/

Steps to deploy:

1) Install Hugo
2) Clone Repo
3) Navigate into mdmsite directory
4) run `hugo`
5) copy everything in the "public" directory that was made to the directory on the server
